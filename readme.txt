== Description ==

This plugin used for add automatically 'Play Pause' button on video, which is implemented using 'HTML5 Video' tag and you can add [Sticky and Floating](https://codepen.io/sunny-sehgal/pen/vdaWvX) feature in video on page scroll, when video on playing mode. See the [Live demo](https://codepen.io/sunny-sehgal/pen/vdaWvX).

You can also add [Sticky and Floating](https://codepen.io/sunny-sehgal/pen/KQBoWZ) feature on Iframe video of Youtube & Vimeo. See the [Live demo](https://codepen.io/sunny-sehgal/pen/KQBoWZ).

= Features  =

*  Extremely easy to use

*  Responsive

*  Automatic add button to uploaded video

*  On hover button will be show

*  Automatic add Sticky & floating feature on video & iframe (Youtube & Vimeo).

== Installation ==

1. Upload myplug folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Its automatically work on the uploaded vedio.
